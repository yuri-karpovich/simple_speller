# YandexSpeller

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'yandex_speller'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install yandex_speller

## Usage

```ruby
YandexSpeller.fix('Tom Cruize') # => "Tom Cruise"

```
